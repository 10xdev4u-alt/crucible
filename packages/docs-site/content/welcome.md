# Welcome to Crucible

Crucible is a **multi-agent code review orchestrator**. It runs several specialized LLM-powered reviewers in parallel on your code changes and aggregates their findings through a consensus scorer.

## Where to go next

**New to Crucible?** Start with [Getting started](/docs/getting-started).

**Setting up CI?** See [Cookbook → GitHub Actions](/docs/cookbook#recipe-github-actions-with-pr-comment).

**Writing custom agents?** See [Writing custom agents](/docs/writing-agents).

**Migrating from another tool?** See [Migration guide](/docs/migration).

**Building with the library?** See [Library API](/docs/api).

**Looking for something specific?** Try the search bar (top-left).

## What you get

- **13 specialized review agents** out of the box
- **5 LLM providers** (Anthropic, OpenAI, Gemini, Bedrock, Ollama) + OpenAI-compatible
- **8 output formats** (text, JSON, SARIF, Markdown, HTML, JUnit, CSV, GitLab)
- **Pluggable cache** (memory or file-backed)
- **Multi-perspective consensus** with weighted scoring
- **15+ CLI commands** for every workflow
- **TypeScript strict**, 100% typed, 400+ tests
- **Production-grade** with retries, circuit breaker, rate limiting, timeouts
- **Plugin system** for custom agents
- **Open source** under MIT
